# 📊 Sales Performance Analysis Dashboard

## 📌 Project Overview

This project analyzes retail sales data using Excel, Python, MySQL, and Power BI to uncover business insights and build an interactive dashboard.

---

## 🛠 Tools Used

- Microsoft Excel
- Python
- Pandas
- Matplotlib
- MySQL
- Power BI

---

## 📂 Project Workflow

1. Data Cleaning using Excel
2. Data Analysis using Python
3. Exploratory Data Analysis (EDA)
4. SQL Business Analysis
5. Interactive Dashboard using Power BI

---

## 📊 Key KPIs

- Total Sales
- Total Profit
- Total Orders
- Total Quantity
- Profit Margin

---

## 📈 Dashboard Insights

- Sales by Category
- Profit by Category
- Monthly Sales Trend
- Sales by Region
- Sales by Segment
- Top 10 Products
- Sales vs Profit Analysis

---

## 🗄 SQL Analysis

Business SQL queries include:

- GROUP BY
- Aggregate Functions
- HAVING
- CASE Statements
- Window Functions
- Top Products Analysis
- Regional Sales Analysis
